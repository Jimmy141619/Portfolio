# 🔐 CITS2006 Cybersecurity Portfolio

## 📌 Overview

This repository documents a series of practical cybersecurity activities completed as part of CITS2006. The goal of this portfolio is to explore and analyse real-world security concepts across physical environments, digital systems, and human interactions.

The activities cover areas such as:

* Physical security
* Cryptography
* Privacy techniques
* Vulnerabilities and exploits
* Security tools and solutions
* AI and cybersecurity

---

## 🎯 Objectives

* Identify and analyse real-world security mechanisms
* Understand vulnerabilities and risks in different environments
* Apply cybersecurity concepts in practical scenarios
* Develop a security-focused mindset through observation and experimentation

## 🧠 Methodology

Each activity follows a structured approach:

1. **Observation / Research**
2. **Identification of Security Concepts**
3. **Analysis of Strengths and Weaknesses**
4. **Risk Assessment**
5. **Recommendations / Improvements**

---

## 🚀 Key Takeaway

Security is not just technical — it is a combination of physical controls, human behaviour, system design, and continuous monitoring. This portfolio reflects a layered approach to understanding and improving security.
